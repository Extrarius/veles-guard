# Security Policy / Политика безопасности

## Русский

**AI Agent Security Handbook** — образовательный проект о защите AI-агентов. Материалы описывают угрозы и контрмеры для проектирования и ревью, а не для несанкционированного доступа к чужим системам.

### Сообщить о проблеме

- Предпочтительно: [GitHub Security Advisories](https://github.com/Extrarius/veles-guard/security/advisories/new), если доступны для репозитория.
- Альтернатива: issue с **минимальным безвредным PoC** — без рабочих эксплойтов против чужих систем и без инструкций по эксплуатации чужой инфраструктуры.

### Что не публикуем в репозитории

- реальные секреты, ключи, токены, внутренние URL;
- working exploits, направленные на атаку чужих систем;
- дампы offensive payload без defensive контекста;
- материалы, которые напрямую облегчают злоупотребление без согласованного образовательного framing.

### Для contributors

См. [CONTRIBUTING.md](CONTRIBUTING.md). Любой PR с security-sensitive контентом проходит строгое ревью.

## English

**AI Agent Security Handbook** is an educational project on securing AI agents. Materials describe threats and controls for design and review — not for unauthorized access to third-party systems.

### Report a security concern

- Preferred: [GitHub Security Advisories](https://github.com/Extrarius/veles-guard/security/advisories/new), if available for this repository.
- Alternative: an issue with a **minimal harmless PoC** — no working exploits against third-party systems and no instructions for attacking someone else's infrastructure.

### What we do not publish in this repository

- real secrets, keys, tokens, or internal URLs;
- working exploits aimed at attacking third-party systems;
- offensive payload dumps without defensive context;
- material that directly enables misuse without agreed educational framing.

### For contributors

See [CONTRIBUTING.md](CONTRIBUTING.md). Any PR with security-sensitive content undergoes strict review.
